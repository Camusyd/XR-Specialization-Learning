# 🏛️ XR Museum Guide

Una guía de museo en **Realidad Aumentada (AR)** desarrollada con **A-Frame** y **AR.js**.  
Permite escanear una imagen de la Mona Lisa para visualizar un modelo 3D acompañado de narración.
